JavaScript

JavaScript adalah bahasa skrip yang digunakan untuk membuat konten halaman web dinamis, berfungsi untuk membuat elemen yang mampu meningkatkan interaksi pengunjung seperti menu drop-down, animasi, dan warna background dinamis. 

              
                document.getElementById("memulai").innerHTML = "Hello World"

Sejarah JavaScript dimulai pada tahun 1995, diciptakan oleh Brendan Eich di Netscape Communications. Awalnya, JavaScript hanya digunakan untuk keperluan internal dengan web browser perusahaan tersebut, Netscape Navigator.

Awalnya, JavaScript bernama LiveScript, yang kemudian diubah menjadi JavaScript agar bisa menjadi “teman” bagi bahasa pemrograman Java milik mitra mereka, Sun Microsystems.

Statement

Statement atau pernyataan pada javascript adalah suatu instruksi yang di tulis dalam program untuk menjalankan suatu program.

Pada JavaScript statement terdiri dari Nilai, Operator, Ekspresi, Kata Kunci, dan Komentar.

Contoh

              
                let a, b, c;
                a = 5;
                b = 6;
                c = a + b;

Syntax

Sintaks JavaScript adalah seperangkat aturan, pola umum agar Program JavaScript dapat dibangun.

Sintaks JavaScript mendefinisikan dua jenis nilai, yaitu Nilai tetap dan Nilai variabel.
   Nilai tetap disebut Literal.
   Nilai variabel disebut Variabel .

Contoh

              
                  var x, y;
                  x = 5 + 6;
                  y = x * 10;
                  document.getElementById("demo").innerHTML = y;

Variabel

Variabel adalah wadah untuk menyimpan data (menyimpan nilai data).

Di dalam JavaScript, sebuah variabel bisa di isi dengan data apa saja, apakah itu teks, angka, atau data lain yang lebih kompleks seperti object.

Jika yang di tetapkan adalah aturan umum dan fixs maka deklarasikan variabel dengan const. Jika nilai variabel dapat berubah, gunakan let.

Contoh

              
                const var1 = 5;
                const var2 = 6;
                let total = var1 + var2;

Operator

Operator pada Javascript adalah mekanisme yang bisa digunakan untuk mengeksekusi berbagai jenis perhitungan, kalkulasi dan perbandingan di dalam sintaksis Javascript.

Pada dasarnya terdapat 6 jenis operator Javascript. Tipe-tipe operator yang umum dipakai dalam Javascript adalah:
Aritmatika
Penetapan
String
Perbandingan
Operator logika
Operator bersyarat

Contoh

              
                let a = 3;
                let x = (100 + 50) * a;

Tipe Data

Tipe Data adalah pembagian jenis data yang memberitahu program bagaimana sebuah data seharusnya diperlakukan.

JavaScript, terdapat dua klasifikasi yakni Primitif dan Object
Primitif: Tipe data yang sederhana dan hanya memiliki satu jenis tipe data.
Object: Merupakan tipe data yang kompleks dan dapat memuat berbagai jenis tipe data.

Contoh

              
                let x;       // undefined
                x = 5;       // Variabel x adalah Number
                x = "John";  // Variabel x adalah String

Objek

Objek dalam Javascript didefinisikan sebagai mutable properties collection, yang artinya adalah sekumpulan properti (ciri khas) yang dapat berubah nilainya.

Objek Javascript terdiri dari banyak properti. Masing-masing properti ini digunakan untuk mendeskripsikan objek tersebut. Sebuah properti idealnya berhubungan langsung dan mendeskripsikan objek dengan jelas.

Contoh

              
                const User01 = {
                  namaDepan: "Bima",
                  namaBelakang: "Maarschal",
                  umur: 21,
                  tinggiBadan: "175"
                };

                document.getElementById("data").innerHTML =
                User01.namaDepan + " is " + User01.umur + " years old.";
