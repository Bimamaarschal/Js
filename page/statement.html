Statement

Statement atau yang dapat di terjemahkan sebagai sebuah pernyataan, yaitu instruksi pemrograman yang akan di eksekusi oleh komputer. Dalam program JavaScript juga terdapat statement/pernyataan pemrograman dari setiap kode yang dibuat.
Berikut contoh sederhana dari statement pada JavaScript: 

              
                let a, b, c;
                a = 5;
                b = 6;
                c = a + b;

Terlihat pada code di atas setiap variabel di isi sebuah data yang berbeda, pada dasarnya contoh di atas merupakan statement pada JavaScript.

JavaScript

Berikut contoh statement pada JavaScript, klik run untuk menampilkan hasil dari program yang anda buat.

Kata Kunci

Pada JavaScript program sering dimulai dengan kata kunci untuk mengidentifikasi tindakan JavaScript yang akan dilakukan.

Kata Kunci Keterangan
Var Mendeklarasikan sebuah variabel
let Mendeklarasikan variabel blok
const Mendeklarasikan konstanta blok
if Menandai blok pernyataan yang akan dieksekusi pada suatu kondisi
switch Menandai blok pernyataan yang akan dieksekusi dalam kasus yang berbeda
for Menandai blok pernyataan yang akan dieksekusi dalam satu lingkaran
function Mendeklarasikan sebuah fungsi
return Keluar dari fungsi
try Menerapkan penanganan kesalahan ke blok pernyataan
