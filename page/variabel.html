Variabel

Variabel adalah wadah untuk menyimpan data (menyimpan nilai data).

Di dalam JavaScript, sebuah variabel bisa diisi dengan data apa saja, apakah itu teks, angka, atau data lain yang lebih kompleks seperti object. Jika ingin membuat aturan umum dan fixs selalu deklarasikan variabel dengan const. Jika nilai variabel dapat berubah, gunakan let. 

              
                const variabel1 = 5;
                const variabel2 = 6;
                let total = variabel1 + variabel2;

Untuk memberi nilai ke dalam variabel, pakai operator assignment menggunakan tanda sama dengan ” = “, seperti pada contoh di atas.

Proses pembuatan variabel dikenal dengan istilah deklarasi.

Proses memberikan nilai awal ke sebuah variabel disebut juga dengan istilah inisialisasi.

Indentifikasi

Identifikasi penggunaan kata kunci variabel pada JavaScript.

              
                const var1 = 5;
                const var2 = 6;
                let total = var1 + var2;

Dari contoh diatas terlihat bahwa kedua variabel var1 dan var2 dideklarasikan dengan const kata kunci. Ini adalah nilai konstan dan tidak dapat diubah.

Variabel (total) dinyatakan dengan let kata kunci ini adalah nilai yang bisa diubah.
