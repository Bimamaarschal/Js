Operator

Operator pada Javascript adalah mekanisme yang bisa digunakan untuk mengeksekusi berbagai jenis perhitungan, kalkulasi dan perbandingan di dalam sintaksis Javascript.

Pada dasarnya terdapat 6 jenis operator Javascript. Tipe-tipe operator yang umum dipakai dalam Javascript adalah:

Operator Aritmatika

Operator Aritmatika digunakan untuk melakukan aritmatika pada angka.

Operator Keterangan
+ Addition
- Subtraction
* Multiplication
** Exponentiation
/ Division
% Modulus (Division Remainder)
++ Increment
-- Decrement

Operator Penugasan

Operator penugasan menetapkan nilai ke variabel JavaScript. Operator Penugasan Penambahan ( +=) menambahkan nilai ke variabel.

Operator Contoh
= x = y
=+ x += y
-= x -= y
*= x *= y
/= x /= y
%= x %= y
**= x **= y

Operator Perbandingan

Operator Keterangan
== equal to

!== not equal value or not equal type
> greater than
< less than

<= less than or equal to
