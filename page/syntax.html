Syntax

Sintaks JavaScript adalah seperangkat aturan, pola umum agar Program JavaScript dibangun.

Sintaks JavaScript mendefinisikan dua jenis nilai, yaitu Nilai tetap dan Nilai variabel.
   Nilai tetap disebut Literal.
   Nilai variabel disebut Variabel . 

              
                var x, y;
                x = 5 + 6;
                y = x * 10;
                document.getElementById("demo").innerHTML = y;

Indentifikasi

Pengidentifikasi adalah nama JavaScript.

Pengidentifikasi digunakan untuk memberi nama variabel dan kata kunci, dan fungsi. Aturan untuk nama resmi sama di sebagian besar bahasa pemrograman.

Nama JavaScript harus diawali dengan:
   Surat (AZ atau az).
   Tanda dolar ($).
   garis bawah (_). 

              
                let variabel1, Variabel2, _variabel3;
                variabel1 = "Bima";
                Variabel2 = "Maarschal";
                _variabel3 = "Rizky";
