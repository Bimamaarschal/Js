Objek

Objek dalam Javascript didefinisikan sebagai mutable properties collection, yang artinya adalah sekumpulan properti (ciri khas) yang dapat berubah nilainya.

Objek Javascript terdiri dari banyak properti. Masing-masing properti ini digunakan untuk mendeskripsikan objek tersebut. Sebuah properti idealnya berhubungan langsung dan mendeskripsikan objek dengan jelas.

Contoh

              
                const User01 = {
                  namaDepan: "Bima",
                  namaBelakang: "Maarschal",
                  umur: 21,
                  tinggiBadan: "175"
                };

                document.getElementById("data").innerHTML =
                User01.namaDepan + " is " + User01.umur + " years old.";

Terlihat dari contoh diatas objek User01 memiliki properti namaDepan dan namaBelakang karena properti-properti tyersebut berguna untuk mendeskripsikan data dasar dari User 01.
